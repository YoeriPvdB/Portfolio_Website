OCDoom is a first person hygiene shooter developed by a team of 4, including myself. The brief for this project was to create a game that could teach a concept. We chose to teach cleaning through an FPS. Furthermore the brief demanded that we pay attention to accessibility options which the game also adheres to. I was in charge of programming the game.

Resources

  • Unity Engine
  • C#
  • Adobe Photoshop
  • Github

the player possesses 2 main attack types, a cloth for melee attack, and a spray bottle for ranged. The cloth works against dust bunnies, while the spray bottle is effective against germ enemies. I implemented the player movement, combat mechanics, and enemy behaviour and movement.

I created the accessibility features. The game's controls are fully customisable. Furthermore, the can can be played with one hand using only the mouse, which I developed.
