FONS - Gardens

  • Resources
    • Node.js
    • Javascript
    • MYSQL
    • JQuery
    • Github
  • Description: Garden themed image sharing website
  • Duration: 3 weeks

This project was developed during my Masters. We were tasked with creating a social media website, for which I chose the theme of gardens. Fons, by my understanding is Latin for "Spring". I chose this theme because my mother has put a lot of work into her garden, and I wanted to present it online.

FONS uses Node.js together with MYSQL to operate dynamically. The website allows for the following server interactions:

  • Creating a password protected account
  • Logging in to the account
  • Changing profile picture
  • Uploading a post when logged in
  • Commenting on posts when logged in
  • Liking posts when logged in
  • Logging out

Users can create an account which then allows them to have their own profile page. From here they can upload posts, change their picture, or view their previous posts.

I created 4 database tables to organise the data. These are "users", "posts", "likes", "comments". The server probes these tables for information to update the website dynamically. The posts on the home page can be sorted between most recent or most liked. Furthermore, users are also listed on a seperate tab. Users may also comment on other posts and like them as well. Likes and comments are updated asynchrnously though JQuery Ajax functions.
