In a team of 5, I helped create an information based website about the Dome of the Rock, a historical and religiously significant building in Jerusalem.

My Work

I was in charge of implementation. I built the website, implementing the designs created by other team members. I used Bootstrap for the navbar. The rest was built with HTML, CSS, and Javascript.

The website included some unique features such as a horizontally scrolling timeline, which I implemented.

I placed a lot of effort around creating scalable and easily adaptable code. This was prioritised as team members in charge of adding content would be able to do so independently and easily.
