This is a gamified simulation of viral transmission made in Processing Java. The user can actiovate the virus on a location causing a shockwave. Nodes close to the center of the shockwave are more likely to be infected. Once infected nodes turn red and can transmit the virus to other nodes. The user can activate the virus 3 time per playthrough. The user can also choose on a spectrum between higher infection rates, and higher virus damage.

Resources

Each node has inherent stats including health, immunity, and chance to infect others. If infected, the nodes health decreases. If it reaches below 0, the node dies. However after a certain time, the node can recover from the infection and then carries immunity. The node also turns green. Immunity does not mean it cannot be infected, but the likehood is very slim, around a 5-10 percent chance. Once a node has recovered it also slowly regenerates its health over time.

The user can also start an "event". Doing so cause all nodes in the vicinity to crowd around the spot the user has clicked on. This will allow for more nodes to be infected due to the overcrowding. The user can do this 3 times per play through.
